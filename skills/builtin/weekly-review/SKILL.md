# Weekly review

Collect only verifiable work from the requested period. Separate completed outcomes, in-progress work, blockers, decisions, and next actions.

Use the default format: headline; completed and impact; active work; blockers/risks; decisions needed; next-week priorities. Link evidence or identify the source conversation when available.

Do not turn plans into completed work or fill gaps with plausible activity. Ask for the reporting window if it is ambiguous.
