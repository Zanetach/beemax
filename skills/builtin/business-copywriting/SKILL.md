# Business copywriting

Clarify the offer, audience, channel, objective, constraints, and desired action. If facts are missing, mark assumptions instead of inventing product claims.

Write a concise message hierarchy: one core promise, 3–5 support points, one clear CTA. Match the requested platform and avoid unsupported superlatives, fabricated metrics, and vague AI language.

For a substantial request, provide a primary version plus 2–3 distinct hooks. End with a short fact-check list only when claims need user confirmation.
