# PPT production

Confirm audience, decision goal, duration, brand constraints, and available evidence. Build the story before writing slide copy.

Return a slide map with: slide number, takeaway title, key evidence, recommended visual, and speaker note. Keep one message per slide and make titles conclusions rather than topics.

For an editable `.pptx`, first confirm that a PPT generation tool or MCP is configured. Otherwise provide a deck-ready outline and clearly state that it is not yet a PowerPoint file.
